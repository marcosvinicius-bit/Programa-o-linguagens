Como serão as linguagens de programação em 20 anos

A tecnologia vem se tornado cada vez mais presente em nossas vidas, e a programação é uma habilidade fundamental para quem deseja atuar neste meio mas, antes, é preciso analisar qual linguagem de programação começar pois cada uma contém suas particularidades e aplicações específicas.

Muita coisa pode mudar em 20 anos, inclusive as linguagens para os programadores que é Python e JS entre outros. Com o passar dos tempos as linguagens de programação venham avançando muito, com isso é impossivel dizer ao correto como elas poderão se desenvolver em 20 anos, algumas deixam de existir, mas poderemos explorar muito mais e saber as linguagens de programação, Python pode começar a desenvolver muito e evoluir com o tempo. JavaScript é uma das linguagens de programação mais utilizadas em todo o mundo, sendo usada tanto no front-end quanto no back-end de sites. JavaScript é uma das linguagens mediana e dificil de aprender pois ele é uma linguagem que é mais utilizada nos jogos, web sites e aplicativos.

A programação está sendo usada muito hoje em dia e com o tempo vai se tornando cada vez mais ligado ao nosso mundo, em 20 anos vai ser o ano da tencologia e programadores do mundo inteiro estarão trabalhando para desenvolver e explorar os codigos das linguagens.
